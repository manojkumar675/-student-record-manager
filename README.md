# 🎓 Student Record Manager — MERN Stack

A full-stack web application to manage student records 
with JWT authentication and full CRUD operations.

## 🚀 Live Features
- 🔐 User Registration & Login (JWT Auth)
- ➕ Add Student Records
- ✏️ Edit Student Details
- 🗑️ Delete Students
- 🔍 Search by Name or Course
- 📱 Responsive Design

## 🛠️ Tech Stack
**Frontend:** React.js, React Router, Axios, React Toastify  
**Backend:** Node.js, Express.js, REST APIs  
**Database:** MongoDB Atlas  
**Auth:** JWT, Bcrypt.js  

## ⚙️ Setup Instructions

### Backend
cd backend
npm install
create .env file with:
  MONGO_URI=your_mongodb_atlas_uri
  PORT=5000
  JWT_SECRET=your_secret_key
npm start

### Frontend
cd frontend
npm install
npm start
